이 이론을 초대칭(Supersymmetry)이라고 해요. 수식이 너무 우아해서 "이게 맞을 것 같다"고 느꼈어요. 실험 결과 없이요.

그런데, 이런 일이 전에도 있었어요

"수식이 너무 아름다워서 이게 맞을 것 같다" — 이 직관이 실제로 들어맞은 경우들이에요. 실험보다 수식이 먼저였어요.

디랙 방정식 (1928) 4년 후 확인

"전자 방정식에 상대성이론을 합쳤더니 음의 에너지 해가 나왔어. 전자의 반물질이 존재해야 해."
아무도 반물질을 본 적 없었어요.

→ 1932년 앤더슨, 우주선(cosmic ray)에서 포지트론(반전자) 발견. 디랙이 노벨상.
맥스웰 방정식 (1865) 22년 후 확인

"전기와 자기를 하나의 방정식으로 합쳤더니 파동이 나왔어. 빛의 속도와 정확히 같네 — 전자기파가 있어야 해."

→ 1887년 헤르츠, 실험실에서 전파 발생·검출 성공. 라디오·와이파이의 시작.
아인슈타인 일반 상대성 (1915) 100년 후 확인

"시공간이 휜다면 물결처럼 퍼지는 중력파가 있어야 해." 당시엔 검출 자체가 불가능해 보였어요.

→ 2015년 LIGO, 블랙홀 충돌로 생긴 중력파 첫 검출. 100년 만의 확인.

세 경우 모두 수식의 아름다움 → 예측 → 나중에 실험 확인 순서였어요. 초대칭을 믿는 물리학자들이 포기 못하는 이유가 여기 있어요.

그래서 짝꿍을 찾기 시작했어요

왜일까요? 두 입장이 있어요.

계속 믿는 쪽

"가속기 에너지가 아직 부족해. 더 강한 장비로 찾으면 나올 거야. 디랙도 4년 걸렸고 아인슈타인은 100년 걸렸는데."

회의적인 쪽

"40년이 지났어. 나올 에너지 범위는 다 뒤졌어. 아름다운 수식이 자연과 다를 수도 있어."

토론 질문: 수식이 아름답다는 이유만으로 이론을 계속 믿는 것, 이건 과학인가요 — 아니면 믿음인가요?

코페르니쿠스와 비교해보면

같은 구조의 질문이에요. 한쪽은 결과가 나왔고, 한쪽은 아직이에요.

코페르니쿠스 (1543)

데이터 없이 "태양 중심이 더 단순하고 아름답다"고 주장.

→ 케플러가 나중에 데이터로 확인.

초대칭 (1981~)

데이터 없이 "수식이 너무 아름답다"고 주장.

→ 43년째 데이터가 안 나옴.

둘의 차이는 뭘까요? 코페르니쿠스는 운이 좋았던 걸까요 — 아니면 아름다움이 진짜 신호였던 걸까요?
