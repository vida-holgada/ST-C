프린키피아의 숨겨진 역사 · 1681–1713

뉴턴의 정보 제국

프린키피아는 천재 한 명의 작품이 아니었다. 브라질의 예수회 선교사, 메릴랜드의 노예주, 캔터베리의 장인, 서아프리카 요새의 천문학자 — 수십 명의 관측자가 데이터를 보냈고, 뉴턴은 그것을 선별하고 가공하고 때로는 묻어버렸다.

Principia Mathematica · Isaac Newton · 1687 / 1713

핼리가 직접 발로 뛰며 수집한 데이터 경로. 파리·로마·단치히·아비뇽의 천문학자들, 브라질과 메릴랜드의 식민지 관측자들, 서아프리카 요새의 기록까지 모두 케임브리지로 흘러들었다.

"그의 관측은 로마에서 첼리오가 한 것만큼 훌륭하다" — 플램스티드

뉴턴의 방법론: 데이터는 관측자와 분리될 수 없었다. 인물이 곧 증거였고, 신뢰도 평가는 과학적 판단이자 사회적 판단이었다.

1
관측자 인적 평가
직업·신분·출신·사용 기기를 먼저 확인한다. "프랑스 국왕 파견"은 신뢰 보증, "주머니 기기"는 감점 요인이지만 결과가 모델과 맞으면 통과.
국왕 파견 → 신뢰직접 면담 → 가중치 상승아마추어 기기 → 감점
2
데이터와 모델의 일치 여부
데이터가 모델과 맞으면 → 관측자의 신뢰도가 높아진다. 맞지 않으면 → 관측자의 능력이나 기기를 문제 삼는다.
스토러: 모델 일치 → 인쇄 승인몬타나리: 경로 불일치 → "구름에 가렸다"
3
중간 검증자 네트워크
핼리와 플램스티드가 현장에서 직접 관측자를 인터뷰하거나 동료에게 물어 2차 신뢰 보증을 생성한다.
핼리 직접 방문 → 헤벨리우스 재평가중간자 없는 식민지 → 낮은 가중치
4
최종 판본에서의 선택적 삭제
초고 → 2판(1713) 사이에 수많은 데이터가 삭제·폄하됐다. "작은 차이는 오차 범위 내"라는 표현으로 불편한 수치를 흡수.
배빙턴: 초고 포함 → 최종판 삭제대븐포트: 이름 삭제

많은 경우 뉴턴은 자신의 모델에 맞지 않는 데이터를 배제한 다음, 정보 제공자들의 판단에서 이러한 적대적 태도를 뒷받침할 근거를 찾으려 했다.

— Simon Schaffer, 텍스트 원문 (저자 분석)

프린키피아 제3권 명제 20 — 지구 각지에서 1초 진자의 길이를 측정. 적도에 가까울수록 진자가 짧아진다는 것이 지구 편구(扁球) 형태의 증거였다. 지도의 파란 숫자는 초판(1687) 출처, 빨간 숫자는 후기판(1713)에 추가된 출처다.
